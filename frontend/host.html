La retención no es el costo. Es el porcentaje temporal de cada cobro utilizado para recuperar el adelanto y el costo informado.

Cómo funciona

3 pasos
  1. Recibís capital.El dinero llega antes de tu temporada fuerte.
  2. Seguís cobrando.Conservás la mayor parte de cada payout.
  3. La retención termina sola.Cuando se completa el total pactado, volvés a cobrar el 100%.
