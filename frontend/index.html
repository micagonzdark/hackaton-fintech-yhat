Qué significa y valor recomendado

Es el monto que el comercio quiere recibir hoy. Para una primera oferta responsable, usar 20%–35% del P10 futuro; en Martín, USD 1.500 mantiene aprobación total.

Qué significa y valor recomendado

Es una estimación conservadora del ingreso futuro: el escenario bajo contra el que se calcula recuperabilidad. Usar P10, no promedio. Para Martín, el valor base es USD 5.200.

Qué significa y valor recomendado

Reduce el P10 para probar una temporada peor a la esperada. Usar 0% como base, 20%–30% como stress razonable y 40%+ como escenario severo.

Qué significa y valor recomendado

Es el porcentaje de cada cobro futuro que se retiene hasta completar adelanto + fee. Usar 20%–35%; valores superiores a 40% pueden presionar demasiado el flujo del host.

Qué significa y valor recomendado

Es el costo total que paga el host por adelantar cobros, no una tasa anual. Para adelantos cortos en USD, usar 3%–5% en perfiles aprobados; 6% sólo para líneas piloto o riesgo más alto.

Qué significa y valor recomendado

Es el rendimiento anual exigido por quienes aportan el capital de la cartera. Para una cartera en USD respaldada por cobros visibles, usar 6%–10%; valor base recomendado: 7%. Testear 10%–12% como stress.

Qué significa y valor recomendado

Aumenta o reduce proporcionalmente la probabilidad de default estimada para probar escenarios. Usar 0% como base y +30% a +50% para un escenario adverso prudente.

Qué significa y valor recomendado

Es el porcentaje del adelanto que no se recuperaría si ocurre un default. Con retención de cobros y límites conservadores, probar 25%–40%; valor base recomendado: 35%.

Qué significa y valor recomendado

Es el retorno anual que FlowRent busca agregar al pricing por encima de fondeo y riesgo. Afecta el fee sugerido, no el resultado del fee contractual actual. Usar 2%–5%; valor base recomendado: 3%.

Qué significa y valor recomendado

Representa KYC, pagos, monitoreo, administración y soporte como porcentaje del adelanto. Para un piloto usar 0,5%–1,5%; valor base recomendado: 1%. Debería bajar con escala y automatización.
